FR
Projet : Logiciel de planification et d’optimisation de ressources
Technologies: Python - Gurobi - Programmation graphique avec tkinter

Description: C'est un logiciel de résolution de problèmes de plannification
et de gestion de resources déstinés à plusieurs types d'entreprises.
Il puise de la puissance de la recherche oppérationnelle pour résoudre
grâce à des modèles mathématiques des poblèmatiques curciales tel que:
- Répartition intelligente des horaires d’employés
- Maximisation des profits selon le stock disponible pour l’entreprise
- Optimisation de la localisation d’un réseau bancaire

Installation:
- Clonez le projet.
- Déplacez-vous vers le répertoire du projet.
- (Optionnel) Activez votre environnement virtuel si vous en avez un.
- Exécutez `python -m pip install -r requirements.txt` pour installer toutes les dépendances requises.

Project: Planning and Resource Optimization Software
Technologies: Python - Gurobi - Graphical Programming with Tkinter

Description: It is software designed for solving planning
and resource management problems for various types of businesses.
It leverages the power of operations research to resolve crucial issues
using mathematical models such as:
- Intelligent scheduling of employee shifts
- Maximization of profits based on available stock for the company
- Optimization of the location of a banking network

Installation:
- Clone the project.
- Navigate to the project directory.
- (Optional) Activate your virtual environment if you have one.
- Run `python -m pip install -r requirements.txt` to install all required dependencies.
